# House Price Prediction: Data Analysis and Machine Learning

## Project Overview
This project is a part of the IBM Data Science Professional Certificate, focusing on data analysis and predictive modeling. 

The goal is to analyse housing data from the US and build models to predict house prices based on features, such as number of bedrooms, square footage, and location. 

## What This Project Demonstrates
- Explores real-world housing dataset.
- Cleans and preprocesses data by handling missing values and dropping unnecessary columns.
- Performs **exploratory data analysis (EDA)** to find relationships between features and property price. 
- Builds and evaluates **multiple regression models.**
- Uses **machine learning** techniques for prediction accuracy.

## Skills Demonstrated
Data analysis and Exploration
- Looking at data types and dataset structures.
- Using pandas to generate statistical summaries.
- Identifying correlations between variables.
- Using seaborn and matplotlib to visualise relationships.

Data cleaning and preparation
- Handling missing values with mean imputation.
- Removing columns that are not needed.
- Feature selection and transformation.

Data Visualisation
- Distribution comparisons using boxplots.
- Regression plots to analyse correlations.
- Interpreting visual data.

Machine Learning
- Linear regression.
- Polynomial regression for non-linear relationships.
- Using scikit-learn to create pipelines.
- Using R-squared to evaluate models.
- Preventing overfitting of data.

Model Evaluation

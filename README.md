# House Price Prediction: Data Analysis and Machine Learning

## Project Overview
This project is a part of the IBM Data Science Professional Certificate, focusing on data analysis and predictive modeling. 

The goal is to analyse housing data from the US and build models to predict house prices based on features, such as number of bedrooms, square footage, and location. 

## What This Project Demonstrates
- Explores real-world housing dataset.
- Cleans and preprocesses data by handling missing values and dropping unnecessary columns.
- Performs **exploratory data analysis (EDA)** to find relationships between features and property price. 
- Builds and evaluates **multiple regression models.**
- Uses **machine learning** techniques for prediction accuracy.
